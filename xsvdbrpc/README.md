# BUILD

mkdir bin && cd bin

Read [api](https://github.com/SimbaBlock/sp/blob/master/xsvdbrpc/doc/api.md)
Edit you configure [file](https://github.com/SimbaBlock/sp/blob/master/xsvdbrpc/conf/conf.json)
cd bin && ./xsvdbrpc ../conf/conf.json
